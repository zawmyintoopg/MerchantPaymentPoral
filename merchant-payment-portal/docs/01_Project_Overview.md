# Merchant Payment Portal

## Project Purpose

Merchant Payment portal is a rest api using spring boot for payment user as qr , visa , master and mpu card transaction and settlement process from mpu. and easy to know merchant info and their related terminal.
2. Merchants make sure their business decision about transaction by analysis report
3. Business function monitoring by upper level.


## Technology

- Java 17
- Spring Boot
- Spring Data JPA
- MySql
- Maven
- Postman
- Git
- React for front-end

## Modules

- Merchant Management 
- Merchant Segment Management
- Terminal Management
- Authentication
- Transaction Management

## Project Goal

provide REST APIS for merchant management with a clean layered architecture.
